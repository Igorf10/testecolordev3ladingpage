## Funcionalidades Principais

### Formulário de Cadastro
- Validação de nome completo (nome + sobrenome)
- Validação de email com correção de erros comuns
- Validação de telefone DDD 21 (Rio de Janeiro)
- Detecção de duplicatas
- Email automático para cliente e admin

### Cartão Fidelidade
- 17 validações rigorosas
- Validação de CPF matemática
- Validação de idade (18+)
- Busca automática de CEP (ViaCEP)
- Validação DDD 21 obrigatório
- Celular deve começar com 9
- Email automático de confirmação

### Busca de Lojas
- Busca por CEP
- Integração com iFood
- Links para WhatsApp e Google Maps
- 43+ lojas cadastradas
